Inventory Management Application
--------------------------------

Tech Stack:
----------
<ul>
<li>UI: Angular 12 </li>
<li>Backend: Spring Boot 2.5.0 , Spring Data JPA, MySQL 8.0, Docker</li>
</ul>

<br>

<b>Swagger URL for Rest Api Documentation:</b> http://localhost:8080/swagger-ui/
